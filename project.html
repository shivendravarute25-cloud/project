Aim of the Experiment

To understand and verify Euler's Theorem for Homogeneous Functions and to compute partial derivatives of homogeneous functions of two and three variables.

  • Understand the concept of homogeneous functions and degree of homogeneity.
  • Verify Euler's Theorem: x·(∂f/∂x) + y·(∂f/∂y) = n·f(x, y)
  • Extend verification to three-variable functions: x·(∂f/∂x) + y·(∂f/∂y) + z·(∂f/∂z) = n·f(x, y, z)
  • Compute numerical partial derivatives interactively.
  • Detect the degree of homogeneity of a given function automatically.

1. What is the Degree of a Function?

The degree of a term is the sum of the powers (exponents) of all variables in that term.

x² + y²
Degree = 2
x³y
3 + 1 = 4
x²y²
2 + 2 = 4

Important: If all terms in a function have the same total degree, the function is called a Homogeneous Function.

2. What is a Homogeneous Function?

A function f(x, y) is said to be homogeneous of degree n if, for any scalar t:

f(tx, ty) = tⁿ · f(x, y)

Example: f(x, y) = x² + 2xy + y² is homogeneous of degree 2 because:

f(tx, ty) = (tx)² + 2(tx)(ty) + (ty)² = t²x² + 2t²xy + t²y² = t² (x² + 2xy + y²) = t² · f(x, y) ✓

3. Euler's Theorem — Simple Explanation

If a function is homogeneous, we can relate the function to its partial derivatives using a shortcut formula — this is Euler's Theorem.

If f(x, y) is homogeneous of degree n and has continuous partial derivatives, then:

x · (∂f/∂x) + y · (∂f/∂y) = n · f(x, y)

For three variables f(x, y, z) homogeneous of degree n:

x · (∂f/∂x) + y · (∂f/∂y) + z · (∂f/∂z) = n · f(x, y, z)

4. Step-by-Step Example (First Order)

Given: f(x, y) = x² + y²

Step 1 — Find Degree

Both terms have power 2  →  Degree n = 2

Step 2 — Find Partial Derivatives

∂f/∂x = 2x   |   ∂f/∂y = 2y

Step 3 — Apply Euler's Formula

x · (2x) + y · (2y) = 2x² + 2y²

Step 4 — Compare with n · f(x, y)

n · f(x, y) = 2 · (x² + y²) = 2x² + 2y²

LHS = RHS  ✔ Verified

5. Second Order Formula (Important)

For a homogeneous function of degree n, the second-order Euler formula is:

x² · (∂²f/∂x²) + 2xy · (∂²f/∂x∂y) + y² · (∂²f/∂y²) = n(n–1) · f(x, y)

This is especially useful for degree n = 2:

First order : x·fx + y·fy = 2f Second order: x²·fxx + 2xy·fxy + y²·fyy = 2(2–1)·f = 2f

6. Step-by-Step Example (Second Order)

Given: f(x, y) = x² + y²

Step 1 — Second Derivatives

fxx = 2   |   fyy = 2   |   fxy = 0

Step 2 — Apply Second Order Formula

x²(2) + 2xy(0) + y²(2) = 2x² + 2y²

Step 3 — Compare with n(n–1)·f

n(n–1)·f = 2·1·(x²+y²) = 2x² + 2y²

LHS = RHS  ✔ Verified

7. Common Examples by Degree

Degree 1 (Linear)

f(x, y) = ax + by

x·a + y·b = 1·f(x,y)

Degree 2 (Quadratic)

f(x, y) = x² + xy + y²

x·∂f/∂x + y·∂f/∂y = 2f

Degree 3 (Cubic)

f(x, y) = x³ + y³

x·∂f/∂x + y·∂f/∂y = 3f

Fractional Degree

f(x, y) = √(x² + y²)

x·∂f/∂x + y·∂f/∂y = 1·f

9. Conclusion

Euler's theorem helps to simplify calculations of homogeneous functions by relating them directly to their derivatives. It is very useful in solving differentiation and integration problems quickly — and is a frequently tested result in Engineering Mathematics exams.
